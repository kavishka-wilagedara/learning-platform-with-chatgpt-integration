# Frontend - learning-platform-with-chatgpt-integration

This is the frontend of the learning-platform web application, built with React and Vite.

## Prerequisites

- Node.js (v18 or higher version)

## Setup

1. **Clone the repository**
```
git clone https://github.com/kavishka-wilagedara/learning-platform-with-chatgpt-integration.git
cd learning-platform-with-chatgpt-integration/frontend

```

2. **Install dependencies**
```
npm install

```

3. **Setup environment variables**
```
VITE_API_URL=your_backend_url

```

4. **Run the server**
```
npm run dev

```
